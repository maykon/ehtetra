# Eh Tetra Command Line Sound

:raised_hands::smile: Eh Tetraaaaaaaaaaaaa!

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g ehtetra

## Running
Run command:

    ehtetra

Or import in NodeJS script:

    import ehtetra from 'ehtetra';

    ehtetra();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx ehtetra
